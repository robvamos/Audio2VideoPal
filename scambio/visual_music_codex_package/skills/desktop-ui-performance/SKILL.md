# Skill: Desktop UI Performance

## Purpose

Build a beautiful but lightweight Windows UI.

## Recommended stack

- Tauri;
- React;
- TypeScript;
- Tailwind;
- Framer Motion.

## Guidance

- Keep UI separate from render/encode.
- Virtualize long plugin/session lists.
- Throttle metric updates.
- Preview must be optional.
- Use dashboard cards for FPS, encoder, dropped frames, CPU/GPU.
- Support dark theme, DPI awareness and multi-monitor.
