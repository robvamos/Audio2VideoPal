# Visual Music Codex Package

Questo pacchetto contiene specifiche e materiali per avviare il progetto con Codex.

## Struttura

- `docs/`: specifiche funzionali, architetturali e tecniche.
- `database/schema.sql`: schema SQLite proposto.
- `configs/project_config.json`: configurazione e cataloghi iniziali.
- `skills/`: skill custom suggerite per guidare Codex.
- `prompts/codex_initial_prompt.md`: prompt iniziale da incollare in Codex.
- `docs/06_backlog.md`: roadmap implementativa.

## Prima azione suggerita

Dare a Codex il contenuto di `prompts/codex_initial_prompt.md` e allegare l'intero pacchetto.
