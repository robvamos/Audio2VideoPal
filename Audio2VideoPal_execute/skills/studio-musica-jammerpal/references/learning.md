# Learning

## Due modi di imparare

### Sapendolo

Usa:
- BPM noto
- pause note
- beat `1` noto
- struttura nota

Serve per bootstrap robusto di:
- tempo
- riallineamento
- fase

### Provando e chiedendo conferma

Usa:
- feedback rapido
- valutazione configurazione
- note utente

Serve per adattare:
- soglie
- pesi
- tolleranze
- convergenza musicale

## Cosa salvare

- campione
- configurazione
- BPM dominante
- stabilita'
- readiness
- lock
- accuratezza beat `1`
- numero di resync
- rating
- nota

## Convergenza musicale

Il sistema deve:
- partire non invasivo
- supportare senza sbilanciarsi
- aumentare presenza solo se la confidenza cresce davvero

## Rete neurale

Ha senso come strato superiore, non come sostituto iniziale di:
- preprocessing
- BPM detection
- re-lock di base

Puo' aiutare per:
- struttura del brano
- continuita' ritmica
- indizi armonici o timbrici
- scelta delle configurazioni migliori
